public String(int[] codePoints, int offset, int count) Allocates a new String that contains characters from a subarray of the Unicode code point array argument. The offset argument is the index of the first code point of the subarray and the count argument specifies the length of the subarray. The contents of the subarray are converted to chars; subsequent modification of the int array does not affect the newly created string. Parameters: codePoints - Array that is the source of Unicode code points offset - The initial offset count - The length Throws: IllegalArgumentException - If any invalid Unicode code point is found in codePoints IndexOutOfBoundsException - If the offset and count arguments index characters outside the bounds of the codePoints array Since: 1.5
